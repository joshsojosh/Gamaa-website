# Gamaa - Your Gaming Hub

Gamaa is a personal gaming website that hosts a collection of browser-based games to play when you're bored. The site is built with HTML, CSS, and vanilla JavaScript.

## Features

- Responsive design that works on desktop and mobile devices
- Collection of browser-based games
- Clean and modern user interface
- Easy navigation between games

## Games Included

1. **Snake Game** - The classic snake game where you control a snake to eat food and grow longer without hitting walls or yourself.
2. **Memory Match** (Coming Soon) - A card matching game to test your memory.
3. **Tetris** (Coming Soon) - The classic block-stacking game.

## How to Use

1. Clone this repository
2. Open `index.html` in your web browser
3. Navigate to the Games section and select a game to play
4. Enjoy!

## Adding New Games

To add a new game:

1. Create a new folder in the `games` directory
2. Add your game files (HTML, CSS, JS)
3. Add the game information to the `games` array in `js/games.js`
4. Add a game image in the `images` directory
